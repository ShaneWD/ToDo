# ToDo

### Steps
- ``` virtualenv venv ```
- ``` cd venv ```
- ``` scripts\activate ```
- ``` git init ```
- ``` git clone https://github.com/ShaneWD/ToDo ```
- ``` cd ToDo ```
- ``` pip install -r requirements.txt ```
- ``` python manage.py runserver ```
- Navigate to *http://127.0.0.1:8000/* on the browser
